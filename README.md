## Features

- Dark Theme
- NextJS
- TailwindCSS
- Mobile Friendly

## Installation

Install gitwind-portfolio with yarn

```bash
  yarn install
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/dntandan/gitwind-portfolio
```

Go to the project directory

```bash
  cd gitwind-portfolio
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
